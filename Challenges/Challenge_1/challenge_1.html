Assignment Summary

This project was developed as part of the Interactive Web Application Sprint 4 assignment. Below is a summary of how the requirements were fulfilled:

  • jQuery UI Widgets Used:
    • .tooltip() – Added to answer option buttons
    • .dialog() – Used for the Game Over modal
    • .accordion() – Used for informational sections
    • .progressbar() – Shows current quiz progress with percentage
  • External jQuery Plugin:
    • jquery-confirm – Used for restart confirmation with styled modal
  • Additional Features:
    • Animated feedback after each answer
    • Score tracking and final score presentation
    • Responsive and modular layout

All logic was implemented in a modular and maintainable way using JavaScript, jQuery, and CSS, following best practices. The game is fully functional without requiring a backend or server.

How to Play

Read each programming question carefully and choose the correct answer. Immediate feedback will be shown after each answer. Try to get them all right!

About This Game

This game was built using HTML, CSS, JavaScript, and jQuery. It dynamically loads questions, tracks your score, and provides animated feedback.
